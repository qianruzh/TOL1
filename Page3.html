Example Questions

Mr. Z makes an initial investment of $ 5,000 for a period of 3 years. Find the value of the investment after the three years if the investment earns the annual return of 12 % compounded monthly.

The process to solve this problem would be:

P (Initial value of investment) = $ 5000

i (Rate of return each compound) = it is compounded monthly, so i should be the monthly return rate. Now we only know the annual return rate is 12%, so the monthly return rate should be 12%/12=1%

n (total compounded times) = since it’s compounded monthly, so t should be 3*12 = 36 months in 3 years

A = P(1 +i)^n = 5000*(1+ 1%)^36 = 7,153.84
